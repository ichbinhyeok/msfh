# Opening Protection Estimator Handoff Sheet

Use this sheet after the homeowner returns from the pre-quote brief.

The purpose is not to create a full CRM record. The purpose is to let the estimator decide quickly whether this inquiry is ready for a narrow quote.

## Intake summary
- Homeowner name:
- County / ZIP:
- Contact method:
- Home type:
  - detached
  - attached / townhouse-like
- Report received:
  - yes
  - no
- Recommendation wording copied from report:
- Relevant report page received:
  - yes
  - no
- Opening photos received:
  - yes
  - no

## Likely route
Choose one:

- opening protection is likely the correct first route
- opening protection may be correct, but report wording is still unclear
- attached-home caution is active and must be checked before quoting
- this inquiry is drifting into a broader project than the report currently supports

## Missing items that block a clean quote
Mark any that still need to come in:

- recommendation page missing
- home type not confirmed
- opening photos missing
- county / service area unclear
- homeowner asking for broad package without report support

## Estimator decision
Choose one:

- quote-ready now
- ready for short clarification call
- not quote-ready yet
- redirect to narrower scope explanation before scheduling

## Scope-drift flags
Stop and slow the job down if any appear:

- homeowner wants windows, shutters, doors, and garage door quoted all at once without report support
- attached-home case but homeowner is also asking for roof-heavy work
- quote request sounds like generic storm upgrade shopping instead of report-tied mitigation work
- homeowner assumes reimbursement is already secured

## Internal note to estimator
The job of this handoff is to preserve scope discipline.

Do not treat the packet as approval, reimbursement certainty, or proof that every requested opening belongs in the first quote.
