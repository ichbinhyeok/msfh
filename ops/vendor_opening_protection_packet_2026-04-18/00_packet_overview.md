# Opening Protection Vendor Packet - 2026-04-18

## Why this packet exists
This packet is not a site feature spec.

It is a draft operating packet for Florida opening-protection contractors who deal with My Safe Florida Home homeowners after the inspection report.

The value is simple:

- reduce pre-quote confusion
- cut bad-fit consult time
- keep scope narrow before the first estimate
- make attached-home caveats visible early

## Repeated send moment
Send this packet when all of the below are true:

1. the homeowner says they already have the MSFH inspection report
2. the homeowner wants a quote or wants to know what kind of quote to get
3. the recommendation sounds like opening protection, or the homeowner is not sure whether it is windows, shutters, doors, or a broader package

The send moment is:

`first inquiry received -> before estimator scheduling -> before on-site measurement`

## What is in the packet
1. `01_pre_quote_scope_brief.md`
   - customer-facing one-page brief
   - free candidate

2. `02_estimator_handoff_sheet.md`
   - internal intake summary template
   - vendor-value core asset

3. `03_quote_scope_boundary_sheet.md`
   - front-page quote attachment draft
   - prevents scope drift and later argument

4. `04_first_send_test.md`
   - how to test whether anyone actually uses this packet

## What this packet is not
- not a generic MSFH explainer
- not a directory listing
- not a lead resale asset
- not a CRM
- not a dashboard
- not an approval or reimbursement promise

## Free versus setup split
### Free layer
- public scope brief
- printable PDF version later

### Setup layer later
- vendor branding
- county/service-area tailoring
- intake summary customization
- quote boundary sheet customization
- attached-home language tuned to vendor scope

## What success looks like
The first success is not traffic.

The first success is:

- one vendor sends the packet to one homeowner
- the homeowner comes back more organized
- the estimator can decide faster whether the request is quote-ready

## What to watch for
- if vendors say "just send me leads," the packet has not yet become a painkiller
- if homeowners feel pushed to a contractor too early, the packet loses trust
- if the packet widens scope instead of narrowing it, it fails
