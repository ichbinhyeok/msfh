# Opening Protection Quote Scope Boundary Sheet

Attach this to the front of a draft quote when the homeowner appears to be using the My Safe Florida Home path and the contractor wants to keep the scope narrow.

## What this quote is for
This quote is prepared for the opening-protection scope currently understood from the homeowner's report, photos, and pre-quote intake.

It is meant to help the homeowner review one narrow path first.

## What this quote may include
- the openings currently identified as part of this quote
- product or protection type for those openings
- permit and inspection responsibility, if included
- document or closeout items, if included

## What this quote does not automatically confirm
- program approval
- reimbursement certainty
- that every opening in the home belongs in scope
- that detached-home rules apply to an attached-home case
- that unrelated roof or other upgrade work belongs in this quote

## Items outside this quote unless clearly stated
- unrelated roof work
- optional upgrades
- convenience add-ons
- openings not identified in the intake or report review
- work that would require a different contractor type or a broader project path

## Homeowner verification points
Before signing, the homeowner should verify:

1. this quote matches the recommendation they are acting on
2. the openings in scope are clearly stated
3. optional items are separated from mitigation-path items
4. attached-home cautions have been addressed if relevant
5. permit, inspection, and closeout responsibilities are stated clearly

## Contractor note
This sheet is meant to reduce confusion, not to widen the project.

If the quote starts drifting beyond the report-backed opening-protection path, restate the scope before the homeowner compares price.
